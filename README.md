#### 这套系统是如何工作的？

##### 工作原理很简单：

1.简化陈述并且进行翻译

2.查询已被简化过描述的数据库中的题目

3.删除您搜索的题目

当然，这个方式也不局限于信息学编程问题。您可以使用它来搜索任何类型的文档，**我们也将于 2024 年尝试性开发论文查重功能，敬请期待！**。
